## 2. 主入口：`create_chat_model()`

**文件**：`backend/packages/harness/deerflow/models/factory.py`

```python
def create_chat_model(
    name: str,
    thinking_enabled: bool = False
) -> BaseChatModel:
    """
    从 config.yaml 中的 models[] 列表按 name 查找配置，
    通过反射实例化对应的 LangChain 模型类。
    """
```

**实例化流程**：
1. 从 `get_app_config().models` 按 `name` 查找 `ModelConfig`
2. 解析 `use` 字段为 `(module_path, class_name)`（格式：`module:ClassName`）
3. 动态导入模块，实例化模型类
4. 如果 `thinking_enabled=True`，应用 `when_thinking_enabled` 覆盖配置
5. 将配置中 `$` 开头的值解析为环境变量

---

## 3. 支持的模型提供商

| 提供商 | `use` 字段 | 说明 |
|--------|------------|------|
| OpenAI | `langchain_openai:ChatOpenAI` | GPT 系列 |
| Anthropic | `langchain_anthropic:ChatAnthropic` | Claude 系列 |
| DeepSeek | `langchain_deepseek:ChatDeepSeek` | DeepSeek 系列 |
| Google | `langchain_google_genai:ChatGoogleGenerativeAI` | Gemini 系列 |
| 任意 LangChain | `langchain_xxx:ChatXxx` | 任何 LangChain 兼容提供商 |
| Claude Code OAuth | `deerflow.models.claude_provider:ClaudeChatModel` | 通过 Claude Code 授权 |
| Codex CLI | `deerflow.models.openai_codex_provider:CodexChatModel` | 通过 Codex CLI |
| vLLM | `deerflow.models.vllm_provider:VllmChatModel` | 自部署 vLLM 端点 |

## 4. config.yaml 模型配置示例

### 标准 OpenAI 配置

```yaml
models:
  - name: gpt-4o
    display_name: GPT-4o
    use: langchain_openai:ChatOpenAI
    model: gpt-4o
    api_key: $OPENAI_API_KEY
    max_tokens: 4096
    temperature: 0.7
```

### 开启 Extended Thinking（思维链）

```yaml
models:
  - name: claude-3-7-sonnet
    display_name: Claude 3.7 Sonnet
    use: langchain_anthropic:ChatAnthropic
    model: claude-3-7-sonnet-20250219
    api_key: $ANTHROPIC_API_KEY
    max_tokens: 8096
    supports_thinking: true
    when_thinking_enabled:
      # 开启思维时覆盖这些参数
      temperature: 1.0
      thinking:
        type: enabled
        budget_tokens: 10000
```

### Vision 支持

```yaml
models:
  - name: gpt-4o-vision
    display_name: GPT-4o with Vision
    use: langchain_openai:ChatOpenAI
    model: gpt-4o
    api_key: $OPENAI_API_KEY
    supports_vision: true    # 开启后 view_image 工具会加载
    max_tokens: 4096
```

### OpenAI Responses API

```yaml
models:
  - name: gpt-5-responses
    display_name: GPT-5 (Responses API)
    use: langchain_openai:ChatOpenAI
    model: gpt-5
    api_key: $OPENAI_API_KEY
    use_responses_api: true
    output_version: responses/v1
```

### Claude Code OAuth（本地 Claude Code 授权）

```yaml
models:
  - name: claude-sonnet-4-6
    display_name: Claude Sonnet 4.6 (Claude Code OAuth)
    use: deerflow.models.claude_provider:ClaudeChatModel
    model: claude-sonnet-4-6
    max_tokens: 4096
    supports_thinking: true
```

`ClaudeChatModel` 按以下顺序查找认证：
1. `CLAUDE_CODE_OAUTH_TOKEN` 环境变量
2. `ANTHROPIC_AUTH_TOKEN` 环境变量
3. `CLAUDE_CODE_OAUTH_TOKEN_FILE_DESCRIPTOR`
4. `CLAUDE_CODE_CREDENTIALS_PATH`
5. `~/.claude/.credentials.json`（纯文本）

### Codex CLI（本地 Codex 授权）

```yaml
models:
  - name: gpt-5.4-codex
    display_name: GPT-5.4 (Codex CLI)
    use: deerflow.models.openai_codex_provider:CodexChatModel
    model: gpt-5.4
    supports_thinking: true
    supports_reasoning_effort: true
```

Codex CLI auth 从 `~/.codex/auth.json` 加载。

### vLLM（自部署，Qwen 推理模型）

```yaml
models:
  - name: qwen-thinking
    display_name: Qwen3-32B (vLLM)
    use: deerflow.models.vllm_provider:VllmChatModel
    model: Qwen/Qwen3-32B
    openai_api_base: http://localhost:8000/v1
    openai_api_key: EMPTY
    supports_thinking: true
    when_thinking_enabled:
      extra_body:
        chat_template_kwargs:
          enable_thinking: true
```

## 5. 模型选择（运行时）

通过 `config.configurable` 在每次对话中选择模型：

```python
config = {
    "configurable": {
        "model_name": "claude-3-7-sonnet",  # 使用 config.yaml 中定义的 name
        "thinking_enabled": True,
    }
}
```

---

## 6. 环境变量解析

配置中 `$` 开头的值会自动解析为环境变量：

```yaml
api_key: $OPENAI_API_KEY   # 解析为 os.environ["OPENAI_API_KEY"]
```

## 7. 缺少提供商模块时的提示

如果依赖的 LangChain 提供商包未安装，`create_chat_model()` 会给出可操作的安装提示：

```
ModuleNotFoundError: langchain-google-genai is not installed.
To install it, run: uv add langchain-google-genai
```

## 8. 开发步骤：添加自定义模型提供商

```python
# backend/packages/harness/deerflow/models/my_provider.py
from langchain_core.language_models import BaseChatModel
from langchain_openai import ChatOpenAI

class MyChatModel(ChatOpenAI):
    """自定义模型提供商，继承 ChatOpenAI。"""
    
    def __init__(self, **kwargs):
        # 自定义初始化逻辑
        super().__init__(**kwargs)
    
    # 可以重写 invoke / astream 等方法来添加自定义行为
```

在 `config.yaml` 中使用：
```yaml
models:
  - name: my-model
    use: deerflow.models.my_provider:MyChatModel
    model: my-model-name
    my_custom_param: value
```
